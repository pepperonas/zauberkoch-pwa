# Security Policy

## Sicherheitslücke melden

Bitte **kein öffentliches Issue** für Sicherheitslücken. Stattdessen:

- E-Mail an **martin.pfeffer@celox.io** (Betreff: „Zauberkoch Security"), oder
- GitHub Private Vulnerability Reporting („Report a vulnerability" im Security-Tab)

Bitte beschreibe das Problem, betroffene Komponente und — falls möglich — Schritte zur Reproduktion. Antwort in der Regel innerhalb weniger Tage; bitte gib uns angemessene Zeit für einen Fix, bevor du Details veröffentlichst.

## Supported Versions

Es wird nur der aktuelle `main`-Stand (= Produktion auf zauberkoch.de) unterstützt.
